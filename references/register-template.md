# Register notes

This register file holds context-specific voice notes: patterns voiceprint has noticed about how you write *in this particular register* (e.g. LinkedIn posts, client emails, blog articles). Cross-register rules live in `voice-profile.md` instead.

Register files are **earned, not pre-defined.** A register file exists only when there's enough signal to make it net-positive over your core voice. Empty or near-empty registers actively harm drafts (they impose preconceptions without enough content to steer well), so voiceprint does not auto-create them at trigger-match time. Registers are created either explicitly ("make this a LinkedIn register") or via review when accumulated patterns earn promotion.

Voiceprint appends approved patterns under the sections below during reviews. You can edit anything by hand.

## Openers

How you tend to start. First lines, hooks, opening moves that sound like you.

## Closings

How you tend to end. Sign-offs, last lines, the move you make to close out.

## Avoid

Words, phrases, or moves that don't fit this register, even if they're fine elsewhere.
